# Musializer

## Introduction

This is the documentation for the music application created using raylib.

## Prerequisities

Download the raylib library from homebrew

```sh
brew install raylib
```

Also, get the `raygui.h` header file from [RayGui Header File][raygui-header] and add it into the external directory inside of the source.
